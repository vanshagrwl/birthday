# Birthday Special

A premium birthday experience built with React and Vite.

## Deployment

This repository is configured for Vercel deployment.

### Build command

`npm run build`

### Output directory

`client/dist`

## Local development

Install dependencies and run both client and server locally:

```bash
npm run install:all
npm run dev
```

## GitHub

Push this repository to GitHub and connect it to Vercel for automatic deployments.
